# v0.16.6 — Live-Monitor: Runway wird jetzt überall angezeigt

🇩🇪 **Deutsch**

## Bugfix (Live-Monitor / Webapp)

- **Die Runway fehlte in mehreren Live-Monitor-Ansichten** (Statistiken,
  Heatmap, Piloten-Detail, Live-Benachrichtigungen) — obwohl die
  Runway-Erkennung selbst hervorragend funktioniert (97,5 % Trefferquote über
  407 echte Flüge, Daten-Audit 2026-06-11).
- Ursache: das Anzeige-Feld wurde aus einer praktisch nie gesetzten Quelle
  befüllt statt aus der Runway-Korrelation. Jetzt fällt es auf den
  Korrelations-Treffer zurück → ab diesem Update zeigen alle Ansichten die
  Runway (für neue Flüge).
