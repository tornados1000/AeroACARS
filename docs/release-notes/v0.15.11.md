# v0.15.11 — Hell-Modus: unsichtbarer Text behoben

🇩🇪 **Deutsch**

## Bugfix (Hell-Modus / Theme)

- **Im hellen Design war Text auf mehreren Seiten unsichtbar** — u. a. im
  **VFR/Manual-Start-Dialog**, in der **Landung/Sinkraten-Analyse** und auf den
  **Plan-Karten**. Ursache: diese Bereiche nutzten ein paar CSS-Variablen, die
  **nie definiert** waren — in der Dunkel-Ansicht fiel das nicht auf, im Hellen
  verschwand der Text (heller/dunkler Text auf gleichfarbigem Grund). Jetzt sind
  alle Variablen sauber definiert und passen sich Hell/Dunkel korrekt an.

## Hinweis

Reines Client-Update. Baut auf v0.15.10 auf.

---

🇬🇧 **English**

## Bugfix (light mode / theme)

- **In light mode, text was invisible on several pages** — including the
  **VFR/Manual start dialog**, the **landing/sink-rate analysis**, and the
  **plan cards**. Cause: those areas used a few CSS variables that were **never
  defined** — it went unnoticed in dark mode, but in light mode the text
  disappeared (text colour matched the background). All variables are now
  properly defined and adapt correctly to light/dark.

## Note

Pure client update. Builds on v0.15.10.
