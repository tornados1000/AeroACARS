# v0.15.25 — Live-Karte: vollständiger Track auch nach Update/Neustart im Flug

🇩🇪 **Deutsch**

## Verbesserung (In-App-Live-Karte)

- Aufbauend auf v0.15.24: die geflogene Linie ist jetzt **auch dann vollständig,
  wenn du AeroACARS mitten im Flug aktualisierst oder neu startest**. Bisher
  begann die lokale Track-Aufzeichnung nach einem Neustart bei null, und die
  Karte zeigte teils noch den alten/lückigen Stand.
- **Jetzt holt sich die App beim Fortsetzen eines laufenden Flugs den
  kompletten Track vom Server** (dort liegt er immer vollständig) und zeigt ihn
  ab der ersten Sekunde wieder lückenlos an — egal ob zwischendurch aktualisiert,
  neu gestartet oder im Hintergrund gedrosselt.
- Schlägt der Server-Abruf fehl (offline), bleibt der lokal gespeicherte Track
  erhalten — kein leerer Bildschirm, kein Absturz.
- **Hinweis zu laufenden Flügen:** Wer mitten im Flug auf diese Version
  aktualisiert, bekommt seine **gesamte bisherige Strecke** wieder sauber
  angezeigt (der Server hatte sie durchgehend). Die **Aufzeichnung selbst war
  ohnehin nie betroffen** — sie ist immer vollständig.

## Hinweis

Reines Client-Update — aktualisiert sich automatisch.

---

🇬🇧 **English**

## Improvement (in-app live map)

- Building on v0.15.24: the flown line is now **complete even if you update or
  restart AeroACARS mid-flight**. Previously local track recording started from
  zero after a restart, and the map could still show the old/gappy state.
- **The app now fetches the complete track from the server** (which always has
  it in full) when resuming an in-progress flight, and shows it gap-free from
  the first second — whether you updated, restarted, or were backgrounded.
- If the server fetch fails (offline), the locally stored track is kept — no
  blank screen, no crash.
- **Note for in-progress flights:** if you update to this version mid-flight,
  your **entire route so far** is restored cleanly (the server had it all
  along). The **recording itself was never affected** — it's always complete.

## Note

Pure client update — auto-updates.
