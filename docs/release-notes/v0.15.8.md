# v0.15.8 — Live-Karte: Follow robust, gestrichelte Route, Wegpunkt-Namen

🇩🇪 **Deutsch**

## Live-Karte

- **Follow zentriert jetzt zuverlässig** — auch wenn du weit rausgezoomt hast.
  Die alte Logik konnte „hängen" (Haken gesetzt, Karte folgt trotzdem nicht);
  das ist neu aufgebaut und mit einem Test abgesichert, der genau diesen Fall
  nachstellt.
- **Dein Zoom bleibt beim Folgen erhalten.** Beim laufenden Folgen wird nur noch
  geschwenkt — kein Zurückziehen mehr auf einen festen Zoom. Ein phasen-passender
  Zoom wird nur noch beim **bewussten** Einschalten (Haken / „🎯 Flugzeug")
  gesetzt.
- **Geplante Route gestrichelt** (wie Stratos) — die **geflogene Spur** bleibt
  durchgezogen. Plan vs. Ist auf einen Blick.
- **Geflogene Spur übersteht einen App-Neustart** (vorher war die Linie nach
  einem Neustart/Update mitten im Flug weg).
- **Wegpunkt-Namen** werden an den Punkten angezeigt — aber erst ab einer
  mittleren Zoomstufe, damit die Karte bei rausgezoomt nicht zugekleistert wird.
- **„VA-Verkehr (N)"** zeigt, wie viele VA-Flieger gerade online/sichtbar sind.

## Hinweis

Reines Client-Update — keine Server-Änderung. Enthält die Fixes aus v0.15.6
(Wegpunkte/Route erscheinen zuverlässig, Follow folgt wirklich).

---

🇬🇧 **English**

## Live map

- **Follow now centres reliably** — even when you've zoomed far out. The old
  logic could "stall" (box ticked but the map wouldn't follow); it's rebuilt and
  covered by a test that reproduces exactly that case.
- **Your zoom is kept while following.** Continuous follow only pans now — no more
  snapping back to a fixed zoom. A phase-appropriate zoom is only set when you
  **deliberately** engage follow (tick / "🎯 Aircraft").
- **Planned route is dashed** (like Stratos) — the **flown track** stays solid.
  Plan vs. actual at a glance.
- **Flown track survives an app restart** (previously the line was gone after a
  restart/update mid-flight).
- **Waypoint names** are shown at the dots — but only from a medium zoom level so
  the map isn't cluttered when zoomed out.
- **"VA traffic (N)"** shows how many VA aircraft are currently online/visible.

## Note

Pure client update — no server change. Includes the v0.15.6 fixes (waypoints/route
show reliably, Follow really follows).
