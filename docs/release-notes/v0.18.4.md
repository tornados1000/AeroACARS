## 🇩🇪 AeroACARS v0.18.4 — Aircraft-Scan-Bedienung überarbeitet

Letzte Nachbesserung an der Ordnerauswahl im Aircraft-Scan-Panel (Settings → Aircraft-Scan), direkt aus dem Feedback der letzten Tage.

**Fix: manuell gewählter Ordner wurde ignoriert.** Nach der Einführung des "Ordner wählen…"-Buttons in v0.18.2/v0.18.3 hängte die Suche den gewählten Pfad nur an die volle automatische Suche an, statt ihn stattdessen zu verwenden — ein Klick auf "Flugzeuge suchen" durchsuchte trotzdem wieder die komplette MSFS-Community- bzw. X-Plane-Aircraft-Bibliothek, vermischt mit dem gewählten Ordner. Behoben.

**Neu: zwei getrennte, klare Buttons statt einem, der sein Verhalten ändert.** "Flugzeuge suchen" durchsucht immer die komplette automatisch erkannte Bibliothek. Der neue Button "Diesen Ordner scannen" direkt neben "Ordner wählen…" durchsucht ausschließlich den eingetragenen bzw. ausgewählten Ordner. Beide Buttons sind immer sichtbar und tun immer dasselbe — kein Umschalten mehr nötig.

## 🇬🇧 AeroACARS v0.18.4 — Aircraft-Scan folder handling reworked

Final follow-up to the folder selection in the Aircraft-Scan panel (Settings → Aircraft-Scan), straight from the last few days of feedback.

**Fix: manually chosen folder was ignored.** After the "Choose folder…" button was introduced in v0.18.2/v0.18.3, the search only appended the chosen path to the full automatic search instead of using it exclusively — clicking "Search for aircraft" still rescanned the entire MSFS Community / X-Plane Aircraft library, mixed in with the chosen folder. Fixed.

**New: two separate, clear buttons instead of one that changes behavior.** "Search for aircraft" always searches the full auto-detected library. The new "Scan this folder" button right next to "Choose folder…" searches exclusively the entered/chosen folder. Both buttons are always visible and always do the same thing — no more mode-switching.
