# v0.16.12 — Neue Flugphasen-Logik im Schatten-Modus

🇩🇪 **Deutsch**

## Unter der Haube: Phasen-Engine v2 läuft im Schatten-Modus mit

Keine sichtbare Änderung für euch: Die bisherige Flugphasen-Erkennung
(Boarding → Climb → Cruise → … → Arrived) bleibt unverändert aktiv und
entscheidet weiterhin alles. Parallel läuft ab jetzt eine neue,
fenster-basierte Phasen-Logik im reinen Beobachtungs-Modus mit und
sammelt Daten für die spätere Umstellung.

Warum: Eine Auswertung über 429 echte Flüge zeigte, dass die alte Logik
in rund einem Viertel der Flüge Phasen falsch labelt — typisch:
ATC-Zwischenlevel im Sink- oder Steigflug werden fälschlich als
„Cruise" erkannt. Die neue Engine bewertet die Bewegung über ein
90-Sekunden-Fenster statt aus Einzelwerten und kennt den vollständigen
Übergangs-Graph (z. B. auch Cruise → Climb für unterbrochene
Steigflüge).

- **Nichts ändert sich an euren Flügen** — PIREPs, Scores, Live-Map und
  Phasen-Anzeige nutzen weiterhin die bewährte Logik.
- Wer genau hinsieht: Weicht die neue Sicht ab, zeigt das Cockpit eine
  kleine graue „v2: …"-Notiz neben der Phase. Reine Verifikations-Hilfe,
  ohne Funktion.
- Die Umstellung auf die neue Logik erfolgt erst nach mehreren Wochen
  Datensammlung und Auswertung.

## Hinweis

🇬🇧 **English**

## Under the hood: phase engine v2 running in shadow mode

No visible change for you: the existing flight-phase detection stays
fully in charge. A new, window-based phase engine now runs alongside it
in observe-only mode, collecting data for a future switch-over.

Why: an audit across 429 real flights showed the old logic mislabels
phases on roughly a quarter of flights — typically ATC level-offs
during climb or descent being latched as "Cruise". The new engine
classifies motion over a 90-second evidence window instead of single
ticks and implements the complete transition graph (including the
previously missing Cruise → Climb edge).

- **Nothing changes for your flights** — PIREPs, scores, live map and
  the phase display all keep using the proven logic.
- If you look closely: when the new engine disagrees, the cockpit shows
  a small muted "v2: …" note next to the phase badge. Verification aid
  only, no function.
- The switch-over happens only after several weeks of data collection
  and review.
