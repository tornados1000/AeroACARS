# v0.15.5 — Logbuch + Live-Karte robuster

🇩🇪 **Deutsch**

## Was ist neu

- **Neues Logbuch** (eigener Reiter): deine **geflogenen Flüge** als Liste
  (Datum, Route, Muster, Dauer, Distanz, Landung, Status) mit Summen oben
  (Flüge · Stunden · Distanz · Ø-Landung · Rang). Klick auf einen Flug öffnet
  das **Detail** mit dem **tatsächlich geflogenen Track** auf der Karte, einem
  **Höhenprofil (MSL · AGL · Gelände)** und dem **Fluglogbuch** (Boarding,
  Takeoff, Cruise … Touchdown). Alles **live über die API** — nichts lokal
  gespeichert.
- **„Meine Flüge" heißt jetzt „Briefing"** (eindeutige Abgrenzung zum Logbuch).

## Live-Karte — robuster

- **Follow sperrt nicht mehr ein:** sobald du selbst pannst/zoomst, pausiert das
  Auto-Zentrieren — du kannst frei schauen. Follow bewusst einschalten zentriert
  sofort.
- **Reload-Schutz:** Beim Wieder-Aufsetzen eines Flugs (Sim-Neustart) folgt die
  Karte einer evtl. ungenauen Lade-Position **nicht** mehr — Hinweis „⏳ warte
  auf Sim-Position", bis der Sim sauber liefert.
- **VA-Flieger bewegen sich flüssig:** zwischen den Updates werden sie entlang
  Kurs + Geschwindigkeit weitergerechnet (Live-Eindruck statt Sprüngen).
- **Positions-Anzeige (POS)** in der Leiste.
- **Wegpunkte überleben einen Resume** (die geplante Strecke geht beim Neustart
  nicht mehr verloren).

## Für VA-Owner

- **Reines Client-Update.** Keine Server- oder Modul-Änderung nötig. Logbuch +
  Karte lesen die phpVMS-API live.

---

🇬🇧 **English**

## What's new

- **New logbook** (its own tab): your **flown flights** as a list (date, route,
  aircraft, duration, distance, landing, status) with totals up top (flights ·
  hours · distance · avg landing · rank). Click a flight for the **detail** with
  the **actually flown track** on the map, an **altitude profile (MSL · AGL ·
  terrain)** and the **flight log** (boarding, takeoff, cruise … touchdown). All
  **live via the API** — nothing stored locally.
- **"My Flights" is now "Briefing"** (clearly separate from the logbook).

## Live map — more robust

- **Follow no longer traps you:** as soon as you pan/zoom yourself, auto-centring
  pauses so you can look around. Turning Follow on re-centres immediately.
- **Reload protection:** when resuming a flight (sim restart), the map no longer
  follows a possibly-wrong loading position — shows "⏳ waiting for sim position"
  until the sim delivers cleanly.
- **VA aircraft move smoothly:** between updates they're dead-reckoned along
  heading + speed (live feel instead of jumps).
- **Position readout (POS)** in the bar.
- **Waypoints survive a resume** (the planned route is no longer lost on restart).

## For VA owners

- **Pure client update.** No server or module change needed. Logbook + map read
  the phpVMS API live.
