# v0.15.7 — Karte: gestrichelte Route + neustart-feste Spur

🇩🇪 **Deutsch**

## Live-Karte

- **Geplante Route jetzt gestrichelt** (wie bei Stratos) — die tatsächlich
  **geflogene Spur bleibt durchgezogen**. So unterscheidet man Plan und Ist auf
  einen Blick.
- **Geflogene Spur übersteht einen App-Neustart.** Bisher war die Linie nach
  einem Neustart (oder Update) mitten im Flug weg, weil sie nur im Speicher lag.
  Jetzt wird sie lokal gesichert und beim nächsten Start wiederhergestellt.

## Hinweis

Baut auf v0.15.6 auf (Wegpunkte/Route erscheinen zuverlässig, Follow folgt
wirklich, „🎯 Flugzeug"-Knopf). Reines Client-Update — keine Server-Änderung.

---

🇬🇧 **English**

## Live map

- **Planned route is now dashed** (like Stratos) — the actually **flown track
  stays solid**. Plan vs. actual at a glance.
- **Flown track survives an app restart.** Previously the line was gone after a
  restart (or update) mid-flight because it lived only in memory. It's now saved
  locally and restored on the next start.

## Note

Builds on v0.15.6 (waypoints/route show reliably, Follow really follows, "🎯
Aircraft" button). Pure client update — no server change.
